# jamiehiggins.dev

My personal website and blog that contains all of my links and an archive of all of my posts. 

Created with [Hugo](https://gohugo.io/) and using [PaperMod](https://github.com/adityatelange/hugo-PaperMod) as the base theme.

## Build
```
git clone https://github.com/JayVee732/jayvee732.github.io.git
cd jayvee732
hugo build
```

## Update PaperMod submodule
`git submodule update --remote --merge`
